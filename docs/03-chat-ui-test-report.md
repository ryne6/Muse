## 📝 用户手册（临时）

### 如何使用当前版本

1. **启动应用**
   ```bash
   npm run dev
   ```

2. **创建新会话**
   - 点击左侧 "New Chat" 按钮

3. **切换会话**
   - 点击侧边栏中的任意会话

4. **发送消息**
   - 在底部输入框输入内容
   - 按 Enter 发送（Shift+Enter 换行）
   - 或点击发送按钮

5. **当前限制**
   - AI 响应是 mock 数据（下一版本将集成真实 AI）
   - 消息不会持久化（刷新后丢失）
